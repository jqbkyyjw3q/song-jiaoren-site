# 宋教仁案专题站

多页面历史解释型专题网站，主题为《宋教仁案与民初共和政治的制度困境》。

## 当前阶段
- 已完成 grokapi 第一轮与第二轮研究采料
- 已确定采用“多页面专题 + 共享设计系统”路线
- 已建立初始站点脚手架，便于后续直接进入页面制作

## 站点结构
- `index.html` — 首页 / 专题导览
- `pages/person.html` — 人物篇：宋教仁是谁
- `pages/case.html` — 事件篇：宋案发生了什么
- `pages/system.html` — 制度篇：为什么共和国没有按共和方式运转
- `pages/turning-point.html` — 转折篇：从宋案到共和失衡
- `pages/debate.html` — 争议与边界
- `assets/css/main.css` — 共享样式
- `assets/js/app.js` — 共享交互
- `docs/design-system.md` — 设计系统说明
- `docs/site-architecture.md` — 信息架构与页面职责
- `docs/research-summary.md` — 研究结论整理
- `research/` — grok 原始研究材料
